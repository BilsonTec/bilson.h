C library-bilson.h


The bilson.h library is a collection of algorithmic functions designed to simplify the resolution of common programming problems. This library offers a variety of basic algorithms that can be easily integrated into your C projects, allowing you to focus on the specific logic of your application rather than on the implementation of underlying algorithms.


Syntax:

#Include <bilson.h>

bilson.h Library Functions


avecal


The avecal function is designed to calculate the average of multiple grades. This function takes an array of grades and the number of grades as input parameters and returns the average of these grades.


Example:

#include <stdio.h>
#include <bilson.h>

int main(){

    // Declaration of an array containing student grades.
    int notes[ ] = {10, 15, 20, 12, 18};
    int number_note = 5;
                
    // Calculating the average by calling the 'avecal' function from your library.
    double average = avecal(notes, number_note);

    // Display the calculated average using %g format specifier.
    // %g: Automatically selects the most compact format (either floating-point or scientific notation)
    // for displaying the number, depending on the value. It will use 6 digits of precision by default.
    printf("The average of the grades is: %g\n", average);
    
    return 0;
}

Output

The average of the grades is: 15.00

time

The time function is designed to convert a given number of seconds into hours, minutes, and seconds. This function takes an integer representing the total number of seconds and three pointers to integers where the hours, minutes and seconds will be stored.


Example:

#include <stdio.h>
#include <bilson.h>

int main(){

    int sec1 = 5000;  // The total number of seconds to convert (5000 seconds in this case).
    int hour,  min, sec2; // Variables to store the hours, minutes, and remaining seconds.

     // Call the 'time' function to convert 'sec1' into hours, minutes, and seconds.
    // 'sec1' is the input total seconds, the result will be stored in hour, min, and sec2.
    time(sec1, &hour, &minute, &sec2);

    // Print the converted time in hours, minutes, and seconds
    printf("it's : %d hour %d min %d sec", hour, minute, sec2); // Display the result.

    return 0;
}

Output

it's : 1 hour 23 min 20 sec

pairodd

The pairodd function is designed to determine whether a given integer is even or odd. This function takes an integer as input and returns a boolean value (1 for true, 0 for false) indicating whether the number is even.


Example:

#include <stdio.h>
#include <bilson.h>

int main(){

    int number = 23; // Declare an integer variable with the value 23.

    // Call the 'pairodd' function to check if the number is even or odd.
    if (pairodd(number))
    {
        // If 'pairodd' returns true (even number), print that the number is even.
        printf("%d is even", number);
    }else{
        // If 'pairodd' returns false (odd number), print that the number is odd.
        printf("%d is odd", number);
    }
                
    return 0;  
}

Output

23 is odd

eqnd

The eqnd function is designed to solve a quadratic equation of the form ax2 + bx + c = 0 . It calculates the discriminant and the real roots of the equation, if they exist.


Example:

#include <stdio.h>
#include <bilson.h>

int main() {

    float a = 1, b = -3, c = 2; // Example coefficients for the quadratic equation (ax^2 + bx + c = 0).
    float delta, racine1, racine2; // Variables to store the discriminant and the roots.
                
    // Solve the quadratic equation (find discriminant and roots) using the 'eqnd' function.
    eqnd(a, b, c, &delta, &racine1, &racine2);//'eqnd' calculates delta and the roots (racine1, racine2).
                
    // Display the discriminant value.
    printf("Discriminant: %f\n", delta);

     // Check the discriminant to determine the nature of the roots.
    if (delta < 0) {
        // If the discriminant is negative, there are no real roots.
        printf("No real roots.\n");
    }else if (delta == 0) {
        // If the discriminant is zero, there is one real double root.
        printf("One real double root: %g\n", racine1);
    }else {
        // If the discriminant is positive, there are two distinct real roots.
        printf("Two distinct real roots: %g and %g\n", racine1, racine2);
    }
                
    return 0;
}

Output

Discriminant: 1
Two distinct real roots: 2 and 1

disval, findmax, ocu

Your code contains three functions: one to input values into an array, one to find the maximum value in the array, and one to count the occurrences of this maximum value.


Example:

#include <stdio.h>
#include <bilson.h>

int main() {

    int tailletab = 5; // Example size of the array (5 elements).
    int tab[tailletab]; // Declare an array with 'tailletab' elements
                
    // / Input values into the array using the 'disval' function
    disval(tab, tailletab); // 'disval' takes user input and stores values in 'tab'.
                
    // Find the maximum value in the array using the 'findmax' function.
    int max = findmax(tab, tailletab); // 'findmax' returns the maximum value in 'tab'.
                
     // Count the number of occurrences of the maximum value using the 'ocu' function.
    int occurrences = ocu(tab, tailletab, max);
                
    // Display the results
    printf("The maximum value is: %d\n", max);
    printf("The maximum value occurs %d times.\n", occurrences);
                
    return 0;
}

srtas

The srtas function sorts an array of integers in ascending order using a simple bubble sort algorithm and then prints the sorted array.


Example:

#include <stdio.h>
#include <bilson.h>

int main() {
    int size = 5; // Define the size of the array.
    int tab[size] = {5, 3, 4, 1, 2}; // Initialize the array 'tab' with values.
                
    // Call the 'srtas' function to sort the array 'tab' in ascending order and print the sorted result.
    srtas(tab, size);
                
    return 0;
}

Output

[1, 2, 3, 4, 5]
